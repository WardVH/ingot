# temporal-viz — the golden-record as-of time machine

An interactive visualization of the temporal pass (`lib/ingest/temporal.ex`). Drag a date and watch
medipim entity **422156**'s golden record materialize at the moment its identity becomes resolvable;
a second scene shows the **over-merge guard** flagging a late bridge instead of silently merging.

Astro + one React island, animated with [motion](https://motion.dev). It reimplements **none** of the
engine — it reads a precomputed snapshot the real Elixir engine produces.

## Run

```bash
cd viz
npm install
npm run dev          # http://localhost:4321
```

Other scripts: `npm run build` (static site → `dist/`), `npm run preview`, `npm test` (Vitest).

## Regenerate the data

The browser can't run Elixir, so the data is exported once from the engine. From the **repo root**:

```bash
mix run temporal_export.exs        # writes viz/src/data/temporal.json
```

`temporal.json` is committed (so the site builds without Elixir) **and** regenerable. It's generated —
don't hand-edit it; re-run the export instead. If the fixture or engine changes, re-export and the
viz updates with zero hand-edited data.

## Layout

```
viz/
  src/
    data/temporal.json     # generated by temporal_export.exs (the only engine→browser bridge)
    lib/types.ts           # the JSON shape
    lib/snap.ts            # lastDateAtOrBefore / tickIndexAtOrBefore (+ snap.test.ts)
    components/            # App (tabs) · TimeMachine (scene 1) · OverMergeGuard (scene 2) · GoldenCard
    pages/index.astro      # the single page, hydrates <App client:load />
    styles/global.css
```
